How to save as og-image.png:

  1. Open this file in Chrome/Firefox
  2. Right-click on the image above and select "Inspect"
  3. In DevTools, right-click the <div class="og-image"> element
  4. Select "Capture node screenshot"
  5. Save the file as og-image.png
  6. Move it to public/og-image.png

Alternative: Use a screenshot tool to capture exactly 1200x630 pixels, or use htmlcsstoimage.com to convert this HTML to PNG.
